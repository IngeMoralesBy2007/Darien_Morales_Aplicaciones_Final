# Darien Garcia - Proyecto Final Aplicaciones I

Este es el repositorio correspondiente a la entrega final de la materia Aplicaciones I.

## Estructura del Proyecto

- `backend/`: Código fuente del servidor (FastAPI, Python).
- `frontend/`: Código fuente de la interfaz de usuario (React/Vite).
- `Ejecutar_Nexus.bat`: Script de inicio rápido del proyecto completo.
- `.env`: Archivo de variables de entorno para la configuración.

## Instrucciones de Despliegue

1. Clonar este repositorio.
2. Asegurar que las dependencias estén instaladas (ver requerimientos de backend y frontend).
3. Ejecutar el script `Ejecutar_Nexus.bat` que levanta ambos entornos (Frontend y Backend).
4. El Frontend estará generalmente disponible en el puerto 5173 o similar, y el Backend en el puerto 8000.
